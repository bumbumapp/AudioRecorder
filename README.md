

<b>Features:</b>
- Recording audio
- Playing back recordings
- Make recordings in M4A and WAV
- Settting sample rate and bitrate
- Recording and playing back in background
- Displaying waveform of the recording
- Rename recordings
- Share recordings
- Import audio files
- List recordings
- Add recordings to bookmarks
- Color themes
